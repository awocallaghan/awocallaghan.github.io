About me

My name is Alex O'Callaghan and I'm a web developer from London, England. This is my personal website created to publish blog posts and projects as I develop my programming skills and create new tools.

I'm interested in all areas of software development with a particular interest in web development. I completed BSc Computer Science at the University of Southampton giving me a strong understanding of Object Orientated Programming using Java and exposed me to Functional Programming Languages such as Scheme and OCaml. Furthermore I have a firm grasp of fundamental Computer Science concepts such as algorithmic complexity, computability and data structures. For my final year project I designed and implemented a video analytics web application for university lecturers using Vue.js + Data-Driven Documents (D3), Express, and RethinkDB.

In my free time I have delved into web development experimenting with JavaScript front end libraries/frameworks such as jQuery, Angular, React and Vue.js. I've created npm packages contributing to the open source JavaScript community and familiarised myself with best practices and useful build tools such as Gulp, Grunt, Webpack, and Browserify. I've also got experience working with a range of server-side languages including JavaScript (Node.js), PHP (Laravel, WordPress), C# (ASP.NET MVC).

I have a strong understanding of HTML5 and CSS3. I've used CSS preprocessors such as SASS and LESS and popular frameworks such as Bootstrap and Foundation to create responsive websites. I've completed freelance web development work designing and implementing professional websites to clients' requirements. I've employed the iterative Agile software development methodology of conducting scrums and sprints to burn through the project backlog when working as a software development team. Furthermore, I understand the importance of following Test Driven Development (TDD) and Behaviour Driven Development (BDD) to ensure a project meets all functional and user experience requirements.
