About me

My name is Alex O'Callaghan and I'm currently studying Computer Science at Southampton University. This is my personal website created to publish blog posts and projects as I develop my programming skills and create new tools.

I'm interested in all areas of software development with a particular interest in web development. My University degree has given me a strong understanding of Object Orientated Programming using Java and exposed me to Functional Programming Languages such as Scheme and OCaml. Furthermore I have a firm grasp of fundamental Computer Science concepts such as algorithmic complexity, computability and data structures.

In my free time I have delved into web development experimenting with JavaScript frameworks such as jQuery, Angular, React and Express. I've created npm packages contributing to the open source JavaScript community and familiarised myself with best practices and useful build tools such as Gulp, Grunt, Webpack, Browserify, etc. I've also used Cordova and Ionic to create a cross-platform smartphone application.

I have a strong understanding of HTML5 and CSS3. I've used CSS preprocessors such as SASS and LESS and popular frameworks such as Bootstrap and Foundation to create responsive websites. I've completed freelance web development work designing and implementing professional websites to clients' requirements. I've employed the iterative Agile software development methodology of conducting scrums and sprints to burn through the project backlog. Furthermore, I understand the importance of following Test Driven Development (TDD) and Behaviour Driven Development (BDD) to ensure a project meets all functional and user experience requirements.
