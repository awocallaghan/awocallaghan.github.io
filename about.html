About me

FootyFeeds project

I created a football news aggregator called FootyFeeds to provide an effective and efficient method for football fans to get all the latest headlines relating to their team. I created the backend using a PostgreSQL database to store the news data and an Express webserver to provide a RESTful API and to regularly gather news articles from RSS feeds. The frontend is an Angular single page application which allows users to register an account, filter news by their favourite team, and filter which RSS sources are displayed. The Angular app is open source and you can get the code on GitHub. I also created social media accounts to help promote the website: Twitter, Facebook, and Google+. I also used Google Analytics to track visitor activity and researched Search Engine Optimisation (SEO) techniques ensuring the address was registered with Google, Bing, Yahoo, etc.
